XPath For JSON

W3C XPath syntax evaluator that works on JSON Documents

What is Xpath-For-Json

  • Use Xpath to select nodes within JSONObject and/or JSONArray
  • Add/Delete/Update/Replace JSON Nodes based on regex
  • Use Javascript like syntax to filter nodes
